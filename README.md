
# shop

Node js shop app using ejs templating engine and simulation of payment using stripe 
[Live Demon](https://shop-jsjunkie.herokuapp.com/).
## Installation



```bash
npm install 
```

## Usage
edit nodemon.json to insert the environment variables 
```node
npm run start:dev
```
